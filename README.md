# Grandel Graden Nexus

Professional desktop client for [Ollama](https://ollama.com/) — manage your local LLMs with a modern, native JavaFX interface. Private, lightweight, and powerful.

## Features

- **Chat Interface**: Multi-session, pin/rename/delete, clean message bubbles with Markdown rendering
- **Model Management**: Browse Ollama library, download with progress, view/delete installed models
- **RAG (Retrieval-Augmented Generation)**: Document indexing + query augmentation via LangChain4j
- **Multi-modal**: Image attachment support for vision models
- **Export Chat**: Export conversations as formatted text files
- **Theming**: Full Light/Dark mode, AtlantaFX design

## Tech Stack

Java 17+, JavaFX 17, Gradle 8.5, AtlantaFX, LangChain4j, Flexmark, Jackson

## Quick Start

```bash
git clone https://github.com/grandelagbanou28-gif/Models_of_ollamaApp.git
cd Models_of_ollamaApp
./gradlew run
```

Fonctionnalités

Tout ce qu'il faut pour l'IA locale

Une application complète, pensée pour la performance et la souveraineté de vos données.

🔒

Confidentialité totale

Tout s'exécute sur votre machine. Aucune donnée ne quitte votre disque. Pas de cloud, pas de pistage, pas de fuite.

⚡

Performant & natif

Application JavaFX native, démarrage instantané, utilisation fluide. Consommation mémoire optimisée pour les longs contextes.

🧠

Tous les modèles Ollama

Llama 3, Mistral, Gemma, Phi, DeepSeek… Parcourez, téléchargez et exécutez n'importe quel modèle compatible Ollama.

💬

Chat enrichi

Markdown temps réel, streaming fluide, RAG avec vos documents (PDF, Word, Excel, images). Pièces jointes illimitées.

🎨

Design moderne

Interface soignée avec AtlantaFX, thème sombre, animations fluides, barre latérale intelligente et responsive.

🌍

Multilingue

Interface complète en français, anglais et espagnol. Support i18n étendu pour une expérience localisée.

📁

Gestion avancée

Historique, dossiers, collections intelligentes, corbeille, recherche full-text. Organisez vos conversations sans effort.

🔧

Personnalisable

Paramètres de génération (température, contexte, top-k/p), prompts système, mode JSON, outils — le contrôle total.

📡

RAG intelligent

Indexation locale de vos documents avec recherche sémantique. Posez des questions sur vos fichiers en langage naturel.

100%

Local — zéro cloud

Licence libre

Aperçu

Une expérience de chat pensée pour la productivité

Streaming temps réel, RAG contextuel, gestion multimodale — Graden Models offre tout ce dont vous avez besoin pour interagir avec vos LLMs localement, sans friction.

  • Streaming token par token avec métriques temps réel
  • RAG avec vos documents personnels (PDF, DOCX, images)
  • Mode multimodal — analyse d'images directement dans le chat
  • Barre latérale adaptative avec gestion de sessions

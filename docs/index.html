Fonctionnalités

Tout ce dont vous avez besoin pour l'IA locale

🔒

Confidentialité totale

Tout s'exécute sur votre machine. Vos données ne quittent jamais votre disque. Pas de cloud, pas de pistage.

⚡

Performant & natif

Application JavaFX native, rapide au démarrage, fluide à l'utilisation. Consommation mémoire optimisée.

🧠

Tous les modèles Ollama

Llama 3, Mistral, Gemma, Phi, DeepSeek… Parcourir, télécharger et exécuter n'importe quel modèle supporté par Ollama.

💬

Chat enrichi

Markdown en temps réel, streaming, RAG avec vos documents, pièces jointes (images, PDF, Word, Excel).

🎨

Design moderne

Interface soignée avec AtlantaFX, thème sombre, animations fluides, barre latérale intuitive.

🌍

Multilingue

Interface en français, anglais, espagnol. Support complet de l'internationalisation.

📁

Gestion des conversations

Historique, dossiers, collections intelligentes, corbeille, recherche. Organisez vos échanges.
